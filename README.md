##[Alfred](http://www.alfredworkflow.com)
Alfred功能强大的工作流软件,可以直接进行对文件进行操作,例如拷贝,粘贴等等。也可以像Spotlight一样索引搜索文件,可以直接输入数字作计算器用,可以查字典,可以当剪贴板用,可以直接当终端用,可以进行网络搜索。
安装完成后使用alt + 空格.
